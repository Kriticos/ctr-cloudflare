# ctr-cloudflare
Container responsável por executar o Cloudflare Tunnel (cloudflared), permitindo a exposição segura de serviços internos sem necessidade de abertura de portas no firewall. Utilizado para publicar aplicações internas (como Zabbix, Grafana e Intranet) através da infraestrutura da Cloudflare, com autenticação, TLS e controle de acesso centralizados.
